Making Videos with FFmpeg
Recently, I helped someone to make youtube videos. Originally I used an interface software but the free version has a lot of limitations. Plus, using a command-line software (like ffmpeg) can be faster for certain functions. Here is a very detailed tutorial on making simple videos using ffmpeg (like overlaying music, video and images).

Just do:

git clone https://github.com/helen-poon/ffmpeg_video.git

Then open video_script.ipybn with Jupyter Notebook and you are there. You can also use Github Codespace to run it.

Making an EPUB(in the near future)
I am good at applying knowledge cross-field. Here is another example. I have some articles online I want to read with Kindle. I webscrape them and use my programming skills to convert them to epub.

I need some time to tidy things up and will publish this project later. If you want to find a project to do and increase your ability of self-learning, it is actually a good one. Learn to use python requests for webscraping. Then, learn how an epub works by unzipping any epub. Simply change the suffix epub to zip, then unzip in the command line. Then you can begin.
